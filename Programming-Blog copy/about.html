About

Welcome to my website! I'm a programmer who would like to inspire other people to start programing. At the beginning it will be a little difficult because it is something completely new. But if you can get thorugh the beggining, then it will be easier because once you learn one programming language, then learning the rest is going to be much easier. The way that I started learning was via books. But, later on, I started to use YouTube as a resource. But, if you are a beginner, I recommend you start to learn via a book, because it is more concentrated rather than open-ended, as it is on YouTube. The first language that you should start learning is Java.
